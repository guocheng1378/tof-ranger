# ToF 测距仪

小米 ToF 传感器测距 App，基于 ST VL53Lx dToF 传感器，支持实时距离测量。

**Apple 液态玻璃 UI** · 悬浮底栏 · 实时数据记录 · 深色/浅色主题

## 📱 支持机型

| 机型 | 传感器 | 量程 | 状态 |
|------|--------|------|------|
| 小米 17 Pro Max | dToF (VL53L5CX) | 4000mm | ✅ 主力机型 |
| 小米 14 Pro | dToF (VL53L1X) | 4000mm | ✅ 支持 |
| 小米 13 Pro | dToF (VL53L1X) | 4000mm | ✅ 支持 |
| 小米 12S Ultra | dToF (VL53L1X) | 4000mm | ✅ 支持 |
| 三星 Galaxy S20+ / Ultra | dToF | 2500mm | ⚠️ 可用（type 不同） |
| 其他带 ToF 的安卓机 | dToF | 视传感器 | ⚠️ 需手动适配 |
| 无 ToF 机型 | Proximity | 近/远 | ❌ 仅二值判断 |

> App 会自动探测传感器型号和 type 值，不需要手动配置。

## 📏 测量范围

| 传感器 | 有效量程 | 精度 |
|--------|---------|------|
| VL53L5CX / VL53L1X | 4000mm（4米） | ±1~3mm |
| VL53L0X | 1200mm（1.2米） | ±3% |
| Proximity (降级) | 近/远 二值 | 不可用 |

## 🚀 安装

### 下载 Release APK（推荐）
1. 进入 [Releases](https://github.com/guocheng1378/tof-ranger/releases)
2. 下载最新 `tof-ranger-v*.apk`

```bash
./gradlew assembleDebug
adb install app/build/outputs/apk/debug/app-debug.apk
```

### 基本测量

1. 打开 App，将手机前端（ToF 传感器位置）对准目标
2. 屏幕中央大字显示实时距离（默认 cm 单位）
3. 保持手机稳定，读数自动平滑
4. 底部精度条越满 = 越稳定

### 悬浮底栏

Apple 液态玻璃风格悬浮底栏，四周有边距，圆角胶囊造型。

| 按钮 | 功能 |
|------|------|
| 🔒 | **锁定** — 冻结当前读数，方便走过去查看。再点解锁 |
| ⏸ | **暂停/继续** — 暂停传感器读数 |
| 📏 | **切换单位** — cm → mm → inch 循环 |
| ⋯ | **更多面板** — 展开额外功能 |

### 数据记录

距离卡片下方的记录区域，可实时采集数据并导出：

1. 点击「开始」→ 清零并开始连续记录，按钮变红，显示 `● 记录中`
2. 实时显示已记录条数（如 `128 条数据`）
3. 再次点击「停止」→ 自动弹出 CSV 文件分享

导出的 CSV 格式：
```csv
timestamp_ms,distance_mm,tilt_deg
1713200000000,123.5,2.3
1713200000200,124.1,2.1
```

## ⚙️ 技术细节

### 滤波算法

```
原始值 → 范围检查 → 尖峰抑制 → 中位数(窗口5) → EMA(α=0.4) → 输出
```

- **尖峰抑制**：单次跳变超过 800mm 的值被丢弃
- **中位数滤波**：窗口=5，去除脉冲噪声
- **EMA 平滑**：alpha=0.4，兼顾响应速度和稳定性

### UI 液态玻璃效果

- **GlassCard** — 多层液态玻璃：底部阴影 → 半透明填充 → 顶部高光 → 底部边缘微光 → 边缘光 → 内阴影 → accent 底边
- **GlassButton** — 玻璃按钮：accent 色调 + press 弹性动效
- **悬浮底栏** — 圆角胶囊容器 + 液态玻璃叠加层 + 浮动边距
