# ToF 测距仪

小米 ToF 传感器测距 App，基于 ST VL53Lx dToF 传感器，支持实时距离测量。

**MiUIX 玻璃态 UI** · HyperOS 暗色主题 · 液态玻璃悬浮按钮

## 📱 支持机型

| 机型 | 传感器 | 量程 | 状态 |
|------|--------|------|------|
| 小米 17 Pro Max | dToF (VL53L5CX) | 4000mm | ✅ 主力机型 |
| 小米 14 Pro | dToF (VL53L1X) | 4000mm | ✅ 支持 |
| 小米 13 Pro | dToF (VL53L1X) | 4000mm | ✅ 支持 |
| 小米 12S Ultra | dToF (VL53L1X) | 4000mm | ✅ 支持 |
| 三星 Galaxy S20+ / Ultra | dToF | 2500mm | ⚠️ 可用（type 不同） |
| 其他带 ToF 的安卓机 | dToF | 视传感器 | ⚠️ 需手动适配 |
| 无 ToF 机型 | Proximity | 近/远 | ❌ 仅二值判断 |

> App 会自动探测传感器型号和 type 值，不需要手动配置。

## 📏 测量范围

| 传感器 | 有效量程 | 精度 |
|--------|---------|------|
| VL53L5CX / VL53L1X | 4000mm（4米） | ±1~3mm |
| VL53L0X | 1200mm（1.2米） | ±3% |
| Proximity (降级) | 近/远 二值 | 不可用 |

## 🚀 安装

### 下载 Release APK（推荐）

### 基本测量

1. 打开 App，将手机前端（ToF 传感器位置）对准目标
2. 屏幕中央大字显示实时距离（默认 cm 单位）
3. 保持手机稳定，读数自动平滑
4. 底部精度条越满 = 越稳定

### 底部按钮说明

| 按钮 | 功能 |
|------|------|
| 🔒 | **锁定** — 冻结当前读数，方便走过去查看。再点解锁 |
| ⏸ | **暂停/继续** — 暂停传感器读数 |
| cm | **切换单位** — cm → m → mm → inch 循环 |
| ↺ | **重置** — 清除所有数据重新开始 |
| ⋯ | **更多** — 展开调试、校准、导出、连测 |

### ⋯ 更多面板

点击 ⋯ 展开后可使用：

- **🔍 调试信息** — 显示传感器原始数据、type 值、量程
- **🎯 校准** — 用已知距离校正测量偏差
- **📄 导出** — 连测数据导出为 CSV
- **⏺ 连测** — 连续测量模式，自动记录多个点位

### 校准

如果测量值有固定偏差：

1. 将手机对准一个已知距离的目标（如尺子）
2. 点「⋯」→「🎯 校准」
3. 输入真实距离（cm），点「开始采集」
4. 保持不动 2 秒，自动校准

### 连测模式

用于连续测量多段距离：

1. 点「⋯」→「⏺ 连测」开启
2. 对准第一个目标，保持不动 1.5 秒 → 自动记录（震动反馈）
3. 移动到下一个目标，再次稳定 → 自动记录
4. 点「⏹ 停止」结束，可导出 CSV

### 音量键

按音量键 = 快速锁定当前距离（跟点 🔒 一样）

## ⚙️ 技术细节

### 滤波算法

```
原始值 → 范围检查 → 尖峰抑制 → 中位数(窗口5) → EMA(α=0.4) → 输出
```

- **尖峰抑制**：单次跳变超过 800mm 的值被丢弃
- **中位数滤波**：窗口=5，去除脉冲噪声
- **EMA 平滑**：alpha=0.4，兼顾响应速度和稳定性

### UI 技术

- `GlassCard` — 毛玻璃卡片，半透明底色 + 顶部光泽渐变 + 边缘高光 + 浮动阴影
- `GlassButton` — 液态玻璃按钮，accent 色调玻璃 + 反射光泽 + 发光边缘 + 回弹动效
- 纯黑背景（`#000000`），MiUIX 配色：蓝 `#5AC8FA` / 绿 `#34C759` / 橙 `#FF9F0A`
