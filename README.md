# ToF 测距仪

小米 17 Pro Max 专属 ToF 测距 App，基于 ST VL53Lx dToF 传感器。

## 功能

- **实时测距** — 毫米级精度，1mm 步进显示
- **单位切换** — mm / cm / m / inch 四种单位
- **锁定拍照** — 锁定当前距离并自动截图保存
- **连续测量** — 稳定后自动记录，适合多点测量
- **数据导出** — 一键导出 CSV 到本地
- **实时曲线** — 可视化距离变化趋势
- **晃动检测** — 检测到晃动时自动冻结显示，避免跳数

## 使用说明

### 基本测量
1. 打开 App，将手机对准目标物体
2. 屏幕中央显示实时距离（默认 cm 单位）
3. 底部状态栏显示传感器采样率和量程范围

### 按钮说明

| 按钮 | 功能 |
|------|------|
| 📸 锁定 | 锁定当前距离读数，自动截图保存到相册。再次点击解锁 |
| ⏹ 连测 | 开启连续测量模式，距离稳定 1.5 秒后自动记录 |
| 📄 导出 | 将测量数据导出为 CSV 文件 |
| ⏸ 暂停 | 暂停/恢复实时测量 |
| 🔄 重置 | 清除所有数据，重新开始 |
| 📏 单位 | 切换单位：cm → m → mm → inch |

### 质量指示

| 标识 | 含义 |
|------|------|
| 🟢 高精度 | 标准差 < 3mm，读数非常稳定 |
| 🟡 良好 | 标准差 3~10mm，正常波动范围 |
| 🔴 波动 | 标准差 > 10mm，建议稳持手机或靠近目标 |

### 连续测量模式
1. 点击「⏺ 连测」开启
2. 对准第一个目标，保持不动 1.5 秒 → 自动记录
3. 移动到下一个目标，再次稳定 → 自动记录
4. 点击「⏹ 停止」结束，App 会语音播报记录点数

### 数据导出
- 点击「📄 导出」生成 CSV 文件
- 文件保存在 App 的 Documents 目录
- 格式：`timestamp, distance_mm`

## 精度优化说明

- **无偏移校准** — ToF 传感器输出绝对距离，不需要减偏移量
- **标准中位数滤波** — 去除脉冲噪声，保留真实信号
- **单次 EMA 平滑** — alpha=0.3，兼顾响应速度与稳定性
- **滑动窗口统计** — 所有统计数据基于最近 100 个采样点

- **降级** — Proximity 传感器，仅支持近/远二值判断
