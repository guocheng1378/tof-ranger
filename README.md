# ToF 测距仪

小米 ToF 传感器测距 App，基于 ST VL53Lx dToF 传感器，支持实时距离测量。

## 📏 最大测量范围

| 传感器类型 | 有效量程 | 精度 |
|-----------|---------|------|
| VL53L1X (dToF) | 4000 mm（4米） | ±1~3mm |
| VL53L0X (dToF) | 1200 mm（1.2米） | ±3% |
| Proximity (降级) | 近/远 二值 | 不可用 |

> 实际量程取决于手机型号和传感器。App 会自动从传感器 API 获取量程。

## 🚀 安装

### 方式一：下载 Release APK
1. 进入 [Releases](https://github.com/guocheng1378/tof-ranger/releases)
2. 下载最新版本 `tof-ranger-v*.apk`
3. 传到手机安装（需开启「允许安装未知来源」）

### 方式二：ADB 安装
```bash
./gradlew assembleDebug
adb install app/build/outputs/apk/debug/app-debug.apk
```

### 基本测量

1. 打开 App，将手机前端（ToF 传感器位置）对准目标物体
2. 屏幕中央大字显示实时距离（默认 cm 单位）
3. 保持手机稳定，读数会自动平滑
4. 「原始: xxx → 滤波: xxx」这行可以对比传感器原始值和滤波后的值

### 质量指示说明

| 标识 | 标准差 | 含义 |
|------|--------|------|
| 🟢 高精度 | < 3mm | 读数非常稳定，可以信赖 |
| 🟡 良好 | 3~10mm | 正常波动范围 |
| 🔴 波动 | > 10mm | 建议稳持手机或缩短距离 |

底部的置信度条（`[████████████░░░░]`）：越满越稳定。

### 按钮功能

#### 📸 锁定
- **作用**：锁定当前距离读数，停止实时更新
- **用途**：一个人测量时，锁定后可以走过去看手机读数
- **自动截图**：锁定时会自动截屏保存到相册
- **解锁**：再次点击「🔓 解锁」恢复实时测量

#### ⏹ 连测（连续测量模式）
- **作用**：自动记录多个测量点
- **使用方法**：
  1. 点击「⏺ 连测」开启
  2. 对准第一个目标，保持不动 1.5 秒 → 自动记录（会震动反馈）
  3. 移动到下一个目标，再次稳定 → 自动记录
  4. 点击「⏹ 停止」结束
- **适用场景**：量房间多个墙面、连续测量多段距离

#### 📄 导出
- **作用**：将连续测量的数据导出为 CSV 文件
- **文件位置**：App 的 Documents 目录
- **格式**：`timestamp, distance_mm`
- **用途**：可以传到电脑用 Excel 打开

#### ⏸ 暂停 / ▶ 继续
- **作用**：暂停/恢复传感器读数
- **用途**：暂时不用时暂停，省电；需要时继续

#### 🔄 重置
- **作用**：清除所有数据，重新开始
- **会重置**：统计数据、连续测量记录、锁定状态

#### 📏 单位切换
- **循环切换**：cm → m → mm → inch
- **默认单位**：cm

### 传感器诊断卡片

界面上会显示传感器的详细信息：

```
传感器: vl53l1x
原始单位: mm | 量程: 4000 mm | 分辨率: 1.0 mm
values.length: 1 | 最近原始值: 1235.00
检测范围: 50.0 ~ 1250.0 (前30个)
滤波参数: 窗口=5 alpha=0.4 尖峰=800mm
```

**如何用这个诊断精度问题**：
- 「原始单位」：自动检测传感器输出的是 mm 还是 cm
- 「量程」：传感器报告的最大范围
- 「最近原始值」：传感器直接给的数，没经过任何处理
- 如果「原始值」看着不对（比如对着1米的墙显示 10000），说明传感器报告单位不一致

## ⚙️ 技术参数

### 滤波算法

1. **范围检查**：丢弃超出量程的读数
2. **尖峰抑制**：单次跳变超过量程 20% 的值被丢弃
3. **中位数滤波**：窗口=5，去除脉冲噪声
4. **EMA 平滑**：alpha=0.4，兼顾响应速度和稳定性

### 自动单位检测

App 启动后会观察前 30 个读数：
- 如果最大值 < 100 → 判断传感器输出的是 cm，自动转为 mm
- 如果最大值 ≥ 100 → 判断传感器输出的是 mm，直接使用

### 采样率

- 传感器以 SENSOR_DELAY_GAME 模式运行（约 20~60 Hz）
- UI 刷新限制在 20fps（50ms 间隔），减少跳动

### 环境要求
- JDK 17
- Android SDK
- Gradle 8.5+

## 📱 兼容性

- 需要 Android 手机内置 ToF 传感器
- 小米 17 Pro Max / 小米 14 Pro / 三星 S20+ 等带 ToF 的机型
- 没有 ToF 传感器的手机会降级到 Proximity 传感器（仅能判断近/远）
