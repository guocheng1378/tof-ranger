# ToF 测距仪

小米 ToF 传感器测距 App，基于 ST VL53Lx dToF 传感器，支持实时距离测量。

**Apple 液态玻璃 UI** · 悬浮底栏 · 实时数据记录 · 白色/深色主题

---

## 📱 支持机型

| 机型 | 传感器 | 量程 | 状态 |
|------|--------|------|------|
| 小米 17 Pro Max | dToF (VL53L5CX) | 4000mm | ✅ 主力机型 |
| 小米 14 Pro | dToF (VL53L1X) | 4000mm | ✅ 支持 |
| 小米 13 Pro | dToF (VL53L1X) | 4000mm | ✅ 支持 |
| 小米 12S Ultra | dToF (VL53L1X) | 4000mm | ✅ 支持 |
| 三星 Galaxy S20+ / Ultra | dToF | 2500mm | ⚠️ 可用（type 不同） |
| 其他带 ToF 的安卓机 | dToF | 视传感器 | ⚠️ 需手动适配 |
| 无 ToF 机型 | Proximity | 近/远 | ❌ 仅二值判断 |

> App 自动探测传感器型号和 type 值，无需手动配置。

## 📏 测量范围

| 传感器 | 有效量程 | 精度 |
|--------|---------|------|
| VL53L5CX / VL53L1X | 4000mm（4米） | ±1~3mm |
| VL53L0X | 1200mm（1.2米） | ±3% |
| Proximity (降级) | 近/远 二值 | 不可用 |

### 基本测量

1. 打开 App，将手机前端（ToF 传感器位置）对准目标物体
2. 屏幕中央大字显示实时距离（默认 cm 单位）
3. 保持手机稳定，读数自动平滑
4. 底部精度条越满 → 测量越稳定

### 悬浮底栏

Apple 液态玻璃风格的悬浮胶囊底栏，五层渲染：磨砂底色 → 顶部高光 → 底部微光 → 边缘亮线 → 内发光。

| 按钮 | 功能 | 说明 |
|------|------|------|
| 🔒 | **锁定** | 冻结当前读数，方便走过去查看。再点解锁 |
| ⏸ | **暂停/继续** | 暂停传感器读数，继续恢复 |
| 📏 | **切换单位** | cm → mm → inch 循环切换 |
| ⋯ | **更多面板** | 展开额外功能（重置、调试、校准等） |

### 🎙️ 数据记录

屏幕中间的记录区域，支持两种记录方式：

#### 方式一：音量键快速记录

| 按键 | 操作 |
|------|------|
| **音量下** | 按一次 → 记录**一条**当前距离数据 |
| **音量上** | 按一次 → 锁定/解锁当前读数 |

每按一次音量下键，振动反馈，记录区域实时更新：
- 条数计数器增加（如 `3 条数据`）
- 显示最近记录的距离值（如 `最近: 12.5 cm`）

#### 方式二：连续记录模式

点击记录卡片右侧的 ⏺ 按钮：

1. **开始** → 清零计数，进入连续记录模式（每 200ms 自动采集一条）
2. 按钮变红，显示 `● 记录中`，实时显示时长和条数
3. **停止** → 自动导出 CSV 文件并弹出分享

导出的 CSV 格式：

```csv
timestamp_ms,distance_mm,tilt_deg
1713200000000,123.5,2.3
1713200000200,124.1,2.1
```

## ⚙️ 技术细节

### 滤波算法

```
原始值 → 范围检查 → 尖峰抑制 → 中位数(窗口5) → EMA(α=0.4) → 输出
```

- **尖峰抑制**：单次跳变超过阈值的值被丢弃，沿用上一次有效值
- **中位数滤波**：窗口 = 5，去除脉冲噪声，排序缓冲区预分配（零 GC 压力）
- **EMA 平滑**：alpha = 0.4，兼顾响应速度和稳定性

### 倾斜补偿

加速度计 + 陀螺仪互补滤波（α = 0.98），实时计算设备俯仰角：

- 水平距离 = 斜距 × cos(俯仰角)
- 垂直高度 = 斜距 × sin(俯仰角)

### UI 渲染

**GlassCard** — 6 层液态玻璃卡片：
```
半透明填充 → 顶部高光渐变 → 底部微光 → 边缘描边 → 内阴影 → accent 底边
```
渐变缓存重建，避免每帧 GC。

**GlassButton** — 玻璃按钮：
- 自定义 Canvas 绘制图标（锁定/暂停/标尺/主题/录制/更多）
- 按下弹性动效（scale 0.88 → 1.0 + OvershootInterpolator）

**GlassBarView** — 悬浮底栏液态玻璃背景：
```
磨砂底色 → 顶部 specular 高光 → 底部 rim 微光 → 1px 边缘亮线 → 内部顶部发光
```

### 性能优化

- 所有 `Paint`/`RectF` 预分配，`onDraw` 零对象创建
- 滤波器排序缓冲区复用，高频传感器回调不触发 GC
- 渐变着色器缓存（仅尺寸/主题变化时重建）
- UI 更新节流 50ms（≈20fps），传感器线程与 UI 线程安全隔离
